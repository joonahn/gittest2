# To New Music
* 1300년~1600년 경 : *신을 위한 음악 -> 인간을 위한 음악*
	* Gregorian Chant 부활절 입당송
	* 1320년 경 'Ars Nova' 중세에서 르네상스
	* Philip de vitry motet(작곡 기법의 발전)
* 1600년 경~1900년 경 Common Practice Period *조성음악*
	* 'Le Nuove Musiche' 그리스 연극으로 돌아가자
	* Scarlatti Cantata
	* Mahler Symphony no.8 finale (팽창)
	* Wagner Tristan und Isolde 서곡 (불협화음의 절정)
* 1900년 경 *전통의 파괴*
	* 'Neue Musik' 서양음악의 '팽창'이 극에 달하자 돌파구를 모색
	* 조성의 파괴 : Schoenberg String Quartet no.3
	* 리듬의 파괴 : Stravinsky '봄의 제전'

# Gustav Mahler
* 1~4번 : Lied(독일 가곡)와 관련
* 8번에서 교향곡 장르를 극한으로 몰고 갔다
* 대지의 노래 : 동양 철학의 영향
* 8번 천인교향곡
	* 800년의 시간과 언어를 뛰어넘어 인류에 대한 구원을 노래
	* 1악장 Veni creator spiritus
		* 856년경 쓰여진 라틴 송가를 가사로 사용
		* "우주의 신에 대한 호소" 표현
	* 2악장 '파우스트' 2부 중 마지막 장면(finale)
		* 인류를 통한 구원에의 믿음, 19세기 음악 양식
		* 오페라처럼 독창자들이 파우스트에 나오는 배역을 맡는다.
		* 바그너적인 반음계주의와 주체적 변형이 주를 이룸(바그너 영향)
	
바그너는 오케스트라가 반주의 역할이 아닌 캐릭터의 심정을 표현하게 하였음, 음악은 시녀가 되서는 안된다고 함
연속적인 느낌과 비화성적인 음악
